# AI RAG Chatbot

An AI-powered Retrieval-Augmented Generation (RAG) chatbot built with:

- FastAPI
- Google Gemini API
- ChromaDB
- LangChain
- React (coming soon)

## Features

- Upload PDF documents
- Extract text
- Chunk documents
- Generate Gemini embeddings
- Store embeddings in ChromaDB
- Ask questions about uploaded PDFs

## Tech Stack

Backend
- FastAPI
- LangChain
- Gemini 2.5 Flash
- Gemini Embeddings
- ChromaDB

Frontend
- React (In Progress)
